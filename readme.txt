PROJECT NAME: SAD SPACE


USAGE: 

This is an educational app geared towards children and young adults. Through user interaction and a series of games, the user can learn about the planets of our local solar system.


TECH USED:

-HTML
-CSS
-JAVASCRIPT
-ANGULAR
